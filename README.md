# AI Based Pneumonia Detection Using Chest X-Ray

## Installation

### 1. Clone Repository
```bash
git clone https://github.com/Hitha2/AI-Based-Pneumonia-Detection-Using-Chest-Xray.git
```

### 2. Open Project Folder
```bash
cd AI-Based-Pneumonia-Detection-Using-Chest-Xray
```

### 3. Create Virtual Environment
```bash
python -m venv venv
```

### 4. Activate Virtual Environment

#### Windows
```bash
venv\Scripts\activate
```

### 5. Install Requirements
```bash
pip install -r requirements.txt
```

### 6. Run Application
```bash
python app.py
```

### 7. Open in Browser
```txt
http://127.0.0.1:5000
```
